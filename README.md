# Ali Özcan - Personal Website

A personal portfolio website showcasing the academic and professional work of Ali Özcan, an undergraduate student specializing in Educational Technology and English Language Teaching.

## Technologies Used

- HTML5
- CSS3 (Vanilla CSS with a focus on academic typography and minimalist design)
- JavaScript

## Features

- Responsive design optimized for both mobile and desktop environments.
- Academic portfolio showcasing education at Pamukkale Üniversitesi and Erasmus+ experience at ANS w Koninie.
- Research section highlighting TÜBİTAK projects and conference abstracts.
- Project gallery for TezTok and Solar Quiz.
- Direct links to professional profiles (LinkedIn, GitHub, ProZ) and CV.
- Academic-focused typography using EB Garamond and Inter.

## Project Structure

- `index.html`: Main structure and content of the site.
- `style.css`: Custom styling, layout, and theme definitions.
- `main.js`: Core JavaScript logic.
- `education/`: Image assets for educational institutions.
- `files/`: Document assets including CV and research PDFs.
- `solar-system/`: Assets related to the Solar Quiz project.
- `teztok/`: Assets related to the TezTok project.

## Local Development

To view the website locally, open `index.html` in any modern web browser or use a local development server such as Live Server.
